Projects

Student Report Card System

Crafted a C++ program aimed at creating comprehensive student report cards, capturing vital personal information (name, roll number, class) and subject-specific marks entered by the user. The program efficiently computes the overall percentage, displaying a detailed report encompassing subject-wise marks, total marks achieved, and the student's overall percentage. Notably, this system offers the ability to save the report card as a file on the computer, providing convenience and record-keeping. Moreover, it empowers the generation of multiple report cards, catering to different students, thereby streamlining the process of academic assessment and documentation.

Network Protocol Simulation
